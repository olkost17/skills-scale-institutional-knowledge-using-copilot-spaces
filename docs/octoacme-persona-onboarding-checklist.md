# OctoAcme Persona Onboarding Checklist

Use this checklist to onboard new team members in any defined persona/role.

- [ ] Assigned onboarding buddy/mentor
- [ ] Provided access to required tools/systems
- [ ] Received walkthrough of project management process docs
- [ ] Completed intro/role-specific training modules
- [ ] Added to relevant standups, syncs, and channels
- [ ] Aware of escalation and feedback paths
- [ ] Reviewed responsibilities and success metrics for their role
